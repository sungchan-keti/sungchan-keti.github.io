# Sungchan Choi

Personal academic homepage for **Sungchan Choi**, a researcher at the **Korea Electronics Technology Institute (KETI)**.

## Website

- GitHub Pages: https://sungchan-keti.github.io/

## About

This site provides a brief English introduction to Sungchan Choi's background, research interests, and selected work related to:

- Internet of Things (IoT)
- Embedded software
- Connected systems and interoperability
- Smart environments and applied digital services

## Local Development

This repository is built with **Jekyll** and can be hosted with **GitHub Pages**.

If you want to run it locally:

```bash
bundle install
bundle exec jekyll serve
```

Then open:

```text
http://127.0.0.1:4000
```

The repository is used to maintain and publish the public profile website for Sungchan Choi.
